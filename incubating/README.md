# 孵化中资产 (incubating/)

存放实验性、未正式发布的 skill、工具、模板和功能。

## 规则

1. **不保证稳定性** — 随时可能重构或删除
2. **必须包含 `INCUBATING.md`** — 说明实验目的、预计成熟时间、已知限制
3. **只进不出** — 成熟后通过 promote 流程移入正式目录（skills/、tools/、templates/）

## Promote 流程

1. 在 incubating 目录内充分验证
2. 通过 `/grill-me` 或 `/grill-with-docs` 审查
3. 移入正式目录并更新相关文档
4. 从 incubating 目录移除（或保留历史记录后归档）
